# 使用手册

本文档面向最终使用者，说明如何运行 `GoldPrice`、如何切换数据源，以及如何添加桌面小组件。

## 1. 启动应用

### 使用 Xcode 运行

1. 打开 `GoldPrice.xcodeproj`
2. 选择 `GoldPrice` scheme
3. 为 `GoldPrice` 和 `GoldPriceWidgetExtension` 设置签名 Team
4. 运行到 `My Mac`

### 使用现成的 `.dmg`

1. 双击挂载 `GoldPrice.dmg`
2. 将 `GoldPrice.app` 拖入 `Applications`
3. 首次打开时，如果 macOS 提示来源不明，请在“系统设置 > 隐私与安全性”里放行

## 2. 菜单栏面板

应用启动后，主入口位于菜单栏。

点击菜单栏中的价格，会展开面板。面板中通常包含：

- 当前美元价格 `USD / OZ`
- 当前人民币克价 `RMB / 克`
- 更新时间
- 当前会话涨跌
- 数据源切换按钮
- `详情`、`刷新`、`退出` 操作按钮

## 3. 数据源切换

菜单栏面板支持三种数据源模式：

- `自动`
  - 默认模式
  - 优先使用 `Kitco`
  - `Kitco` 失败时自动回退到 `Gold API`
- `Kitco`
  - 强制只使用 Kitco
  - 如果 Kitco 页面结构变化或网络不可达，会直接报错
- `Gold API`
  - 强制只使用 `api.gold-api.com`

切换数据源后：

- 当前历史图会清空并重新采样
- 应用会立即发起一次新请求
- 你在这次运行中的选择会被记住

## 4. 价格说明

应用主要显示两个价格：

- `USD / OZ`
  - 每金衡盎司黄金的美元报价
- `RMB / 克`
  - 基于 `USD/CNY` 汇率换算得到的人民币克价

如果人民币价格显示为 `--`，通常说明本次请求没有成功拿到 `USD/CNY` 汇率数据。

## 5. 详情窗口

点击菜单栏面板中的 `详情` 按钮，会打开详情窗口。

详情窗口提供：

- 更大的当前报价展示
- 最近 4 分钟的价格走势
- 当前会话涨跌信息
- 当前数据源和最近更新时间

主应用刷新频率为每秒一次，因此详情窗口会比 Widget 更接近实时。

## 6. 添加桌面小组件

先确保主应用至少成功运行过一次，然后再添加 Widget。

### 方法 A：桌面右键

1. 在 macOS 桌面空白处右键
2. 选择“编辑小组件”
3. 搜索“国际金价”或 `GoldPrice`
4. 选择 `small` 或 `medium`
5. 点击添加

### 方法 B：通知中心

1. 打开通知中心
2. 滚动到底部
3. 点击“编辑小组件”
4. 搜索并添加 `GoldPrice`

## 7. Widget 刷新说明

Widget 不是秒级刷新。

当前策略：

- 正常情况下，大约 15 分钟后刷新一次
- 请求失败时，大约 5 分钟后重试

这是 `WidgetKit` 的正常限制，不是应用 bug。

## 8. 常见问题

### Q: 为什么 `Kitco` 模式下会报错？

可能原因：

- Kitco 页面结构变化
- 网络访问异常
- 临时超时

建议先切回 `自动` 模式，或者直接切到 `Gold API`。

### Q: 为什么我看不到 Widget？

请检查：

- 主应用是否至少成功运行过一次
- `GoldPriceWidgetExtension` 是否签名成功
- 两个 target 的 Team 是否一致
- bundle id 是否冲突

### Q: 为什么价格没有刷新？

请检查：

- 网络是否正常
- 当前数据源是否可用
- 是否正在频繁切换数据源

你也可以点击菜单栏面板中的 `刷新` 手动拉取一次。

### Q: 为什么应用不能直接发给别人用？

因为无签名构建通常只适合本机自用。若要给别人安装，需要额外做：

- Developer ID 签名
- notarization 公证
- stapler 回填

具体步骤见 [构建与发布](BUILD_AND_RELEASE.md)。
